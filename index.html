Q 如果在○○系統上要怎麼安裝？

這問題很難又很好回答……先用 Vagrant 開虛擬機

  • 點選右上角的 Fork me 把這份投影片 clone 下來
  • 安裝好 VirtualBox 與 Vagrant
  • 在 Windows 上你會需要用個 Git Bash，或參考這邊教學
  • vagrant up 搞定
  • 這邊用的是 Debian 8 (Jessie)，Ubuntu 上操作通用
